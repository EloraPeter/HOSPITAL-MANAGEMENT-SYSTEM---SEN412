# Clinical Records User Manual (PR #4)

## Adding a Clinical Record
1. Search or select a patient from the Patients list
2. Open the patient's profile
3. Go to **Clinical Records** tab
4. Click **Add New Record**
5. Fill in:
   - Diagnosis
   - Symptoms
   - Treatment/Prescription
   - Notes
   - Vital signs (optional)
6. Click **Save Record**

## Managing Records
- **View**: All records are listed chronologically
- **Edit**: Click the edit icon on any record
- **Delete**: Click the delete icon and confirm

**Note**: Only authorized medical staff can add/edit clinical records.
